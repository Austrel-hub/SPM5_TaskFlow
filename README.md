# TaskFlow

Este proyecto consiste en una aplicación web básica de gestión de tareas, desarrollada como parte de un trabajo del módulo de Front End.

La aplicación permite agregar tareas, editarlas, eliminarlas y cambiar su estado. Además, utiliza almacenamiento local para guardar la información y también incorpora consumo de API con JavaScript.

## Funcionalidades

- Agregar nuevas tareas
- Editar tareas existentes
- Eliminar tareas
- Marcar tareas como pendientes o completadas
- Buscar tareas por texto
- Guardar tareas en localStorage
- Cargar tareas desde una API
- Enviar una tarea de ejemplo a una API
- Mostrar contador regresivo en tareas con fecha límite

## Objetivo del proyecto

El objetivo de este proyecto fue aplicar contenidos vistos en clases, especialmente:

- Programación orientada a objetos en JavaScript
- Uso de ES6+
- Eventos y manipulación del DOM
- Asincronía con JavaScript
- Consumo de APIs
- Uso de localStorage

## Cómo ejecutar el proyecto

1. Descargar o clonar este repositorio
2. Abrir la carpeta en Visual Studio Code
3. Ejecutar el archivo `index.html` en el navegador

También se puede usar la extensión Live Server para visualizarlo de mejor manera.
